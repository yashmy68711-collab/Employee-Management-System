# Python Employee Management System

## About

A simple CLI Employee Management System built using Python.
This application helps users manage employee records by adding, viewing, searching, updating, and deleting employee information.

---

## ✨ Features

✔️ Add new employees
✔️ View all employee records
✔️ Search employee by ID
✔️ Update employee details
✔️ Delete employee records
✔️ Input validation

---

## 🛠️ Tech Used

* Python
* Object-Oriented Programming (OOP)
* Dictionaries
* CRUD Operations

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Choose an option from the menu
5. Manage employee records easily
